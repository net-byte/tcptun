# Usage  

```
Usage of ./tcptun:  
  -k string
        Encrypt key (default "6da62287-979a-4eb4-a5ab-8b3d89da134b")
  -l string
        Local address (default ":1987")
  -s string
        Server address (default ":1080")
  -S Server mode
```
